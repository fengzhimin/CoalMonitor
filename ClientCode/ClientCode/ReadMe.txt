所有控件的ID编号：ID_NAME_DIALOG_CONFIG
ID：为控件的缩写
NAME：为控件的名称
DIALOG：为控件所在的对话框名称
CONFIG：为控件的类型

类介绍：
ClientCodeDlg：主程序对话框
ConfigDlg：配置对话框
Login：登录对话框
ReadCarCard：读取车卡信息对话框
WriteCarCard：写入车卡信息对话框
Serial：读卡器链接对话框
CarCardInfo：车卡信息操作的类(比如线程参数传递和语音程序)


配置文件介绍
[Database connection Info]    数据库配置
[Remember password Info]      密码是否记住
[Serial Config Info]          登录时，是否加载配置菜单
[SerialCom Config Info]       读卡设备的串口号
[BaudRate Config Info]        读卡设备串口的波特率
[ReadAgainRate Config Info]   重复读卡的频率
[Video Config Info]           摄像头配置
PicturePath        抓图存放的路径
VideoPath          录像存放的路径
PictureMode        图片的格式
PictureWidth       图片的宽度
PictureHeight      图片的高度
